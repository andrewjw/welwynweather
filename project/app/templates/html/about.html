About

This website shows the weather data collected from my back garden in Welwyn Garden City.

The weather station we use is made by Smart Weather and we read the data using PyWWS. It measures indoor and outdoor temperature and humidity, wind speed and direction, pressure and it has rain guage.

The weather station used is not a scientific instrument and it has not be calibrated or properly mounted so that data should be taken with a gain of salt.
